# About Project

# Softwares
1. VS Code
2. Python3

# Packages
1. pip install boto3
2. pip install pillow
3. pip install streamlit

# Execution Process
1. Clone the Repo
2. Install all packages listed above
3. Place Access Key and Secret Access Key in the program
4. streamlit run app.py
